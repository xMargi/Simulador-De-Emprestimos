# Simulador de Empréstimos 📊

Simulador de financiamentos com suporte aos sistemas Price e SAC, exibindo a tabela de parcelas e gráficos de evolução do empréstimo.

![Emprestimo](https://github.com/user-attachments/assets/1a58e1ba-0e65-4db6-8009-1ae039d6c8e9)

## 🚀 Tecnologias

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui
- Framer Motion
- React Google Charts

## ✨ Funcionalidades

- Simulação pelo sistema Price (parcelas fixas)
- Simulação pelo sistema SAC (amortização constante)
- Cálculo de amortização, juros mensais e saldo devedor por parcela
- Gráfico de evolução das parcelas ao longo do tempo
- Validação dos campos do formulário
- Interface com efeito glass e fundo de constelação animado

## ⚙️ Como rodar
```bash
npm install
npm run dev
```

## 📐 Sistemas de Amortização

**Price** — parcelas fixas durante todo o financiamento, com juros decrescentes e amortização crescente.

**SAC** — amortização constante, com parcelas decrescentes ao longo do tempo.
